# apb_master_verification.sv
This project focuses on the functional verification of an AMBA Advanced Peripheral Bus (APB) Master using the Universal Verification Methodology (UVM) in SystemVerilog. The objective is to ensure that the APB Master correctly initiates and manages read and write transactions in compliance with the AMBA APB protocol.
The verification environment is designed to be modular, reusable, and scalable, enabling comprehensive validation of the master’s behavior under various operating conditions. The project is implemented and simulated using EDA Playground, making it easily accessible and shareable.
# APB Master Verification using UVM

## Overview
This project presents a UVM-based verification environment for an AMBA APB Master. It validates read and write operations, protocol compliance, and error handling using constrained-random stimulus and functional coverage.

## Features
- UVM-based reusable testbench
- APB protocol compliance checking
- Scoreboard for data integrity
- Functional coverage and assertions
- Support for wait states and error conditions

## Tools Used
- SystemVerilog
- UVM (Universal Verification Methodology)
- EDA Playground (Questa/Riviera)
